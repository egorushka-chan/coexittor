## Описание

Проект состоит из 2-ух частей: бекенда и фронтенда

- Бекенд: ASP.NET Core Web API, с подключением к базе данных (MSSQL) через EF Core.
- Фронтенд: ASP.NET Core MVC (ничего особенного)

### Бекенд
Состоит из 4 частей (по паттерну DDD):

- Domain - сущности и интерфейсы для репозиториев
- Infastructure - реализация доступа к данным через EF Core
- Application - бизнес-логика
- API - контроллеры и точки входа

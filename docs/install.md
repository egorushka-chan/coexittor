## Гайд по запуску
---
Вроде ничего сложного не предполагается:

- Следует в CoExittor.Api копировать файл DEFAULT_appsettings.json в appsettings.json, и указать там строку подключения к вашей базе данных MSSQL.
- Следует в CoExittor.Frontend копировать файл DEFAULT_appsettings.json в appsettings.json.

После этого проекты должны запускаться. 
Работать должно в любой комбинации профилей запуска проектов.

База данных должна создаваться автоматически с таблицами при первом запуске бекенда.
